# Pico-GPT チュートリアル

このチュートリアルでは、`pico-gpt` を読みながら GPT の理論と実装を学ぶ。
あなたが学習の主体で、私は質問・整理・レビュー・ヒント提供を担当する。

## 学び方
各章で次の順に進める。
1. 対象ファイルを読む
2. 自分の言葉で役割を書く
3. 理論とコードの対応を確認する
4. 小さな演習を解く
5. 学習ログを `docs/learning/` に残す
6. 必要なら Zenn Scrap 用メモに圧縮する

## 章一覧
1. `chapter-01-overview.md` — 全体像
2. `chapter-02-attention.md` — Attention とモデル前半
3. `chapter-03-forward-and-generate.md` — Forward / Loss / Generate
4. `chapter-04-tokenizer.md` — Tokenizer
5. `chapter-05-training.md` — Training
6. `chapter-06-inference-ui.md` — Inference / UI
7. `chapter-07-rebuild-and-explain.md` — 再実装と説明

## パートナーとしての役割
- 先に完成答案を出しすぎない
- 理解を深める質問を返す
- 必要なら最小限のヒントを出す
- あなたの説明をレビューして穴を見つける
- 学習ログと Zenn 用メモの整理を手伝う
