# GPT Learning

このディレクトリには、このリポジトリを使った GPT 学習ログを置く。

## 構成
- `curriculum.md`: 章立て
- `tutorial.md`: チュートリアルの入口
- `session-state.md`: 再開用の現在地
- `chapter-01-overview.md`: 全体像
- `chapter-02-attention.md`: attention とモデル前半
- `chapter-03-forward-and-generate.md`: forward / loss / generate
- `chapter-04-tokenizer.md`: tokenizer
- `chapter-05-training.md`: 学習ループ
- `chapter-06-inference-ui.md`: 推論とUI
- `chapter-07-rebuild-and-explain.md`: 再構成と説明
- `zenn-notes.md`: Zenn Scrap / 最終記事の種

## ルール
- 各章は「目的 / 対象ファイル / 理論 / コード読解 / 演習 / 自分の言葉での要約」を含める。
- 演習の答えはすぐ書き切らず、自分で考えた跡を残す。
- Zennに転用できるよう、気づきは短く独立した文で残す。
- セッション終了時は `session-state.md` を更新し、次回の開始点を残す。
